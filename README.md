# 🚀 Employee Management System (JDBC Project)

## 📌 Project Overview

This is a Java-based console application built using **JDBC and MySQL** to manage employee records.
It performs CRUD operations and demonstrates real-world database handling concepts.

---

## 🛠️ Tech Stack

* Java
* JDBC
* MySQL
* Eclipse IDE

---

## ✨ Features

* Add Employee
* View Employees
* Update Employee
* Delete Employee
* Database connection using JDBC
* PreparedStatement for secure queries
* Clean structure using DAO pattern

---

## 📂 Project Structure

src/
└── jdbcprojectdemo/
├── DBUtil.java
├── EmployeeDAO.java
└── EmployeeApp.java

---

## ⚙️ How to Run

1. Clone the repository
2. Open in Eclipse / VS Code
3. Configure MySQL database
4. Update DB credentials in DBUtil.java
5. Run EmployeeApp.java

---

## 📸 Sample Output

Displays employee records in console after performing operations.

---

## 💡 Key Learnings

* JDBC connection handling
* PreparedStatement usage
* CRUD operations
* Exception handling
* Clean code structure using DAO
